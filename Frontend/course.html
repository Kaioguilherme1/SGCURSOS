Curso de Node.js

Carga Horária: 60 horas

Matriculados Atualmente: 50

Programação Web
JavaScript Node.js Express.js MongoDB

O curso de Node.js é projetado para ensinar os fundamentos da programação com Node.js. Você aprenderá a criar aplicativos web do lado do servidor usando JavaScript e o ambiente de tempo de execução do Node.js.

Os tópicos abordados no curso incluem:

  • Introdução ao Node.js
  • Manipulação de arquivos e pastas
  • Criação de servidores web com Express.js
  • Acesso a bancos de dados com MongoDB
  • Autenticação e autorização de usuários

Este curso é adequado para iniciantes que desejam aprender a desenvolver aplicativos web com Node.js. Não são necessários conhecimentos prévios de programação.
